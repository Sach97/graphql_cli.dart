# Graphql-cli dart
A graphql cli tool written in dart for parsing remote graphql schemas. End goal : generate a local schema from introspection endpoint

## Usage
```dart
bin$ dart main.dart "<GRAPHQL_INTROSPECTION_ENDPOINT>"
```
